* ClickListener 인터페이스 구현 및 사용하는 방법 4가지

인터페이스 생성
아래를 보고 동일하게 패키지와 인터페이스 생성
package com.eastflag.listener; public interface ClickListener { public void onClick(); }
1. 인터페이스를 implements하는 클래스 생성
package com.eastflag.anonymous; import com.eastflag.listener.ClickListener; public class q1 { public static void main(String[] args) { /* * 아래 클래스 생성후 이 클래스의 onClick()함수 호출 */ } } /* * ClickListener 인터페이스를 implements하는 클래스 생성 * onClick()함수는 "Clicked"라는 스트링을 출력하도록 구현한다. */
2. Anonymous class 문법으로 생성하고 onClick 호출
package com.eastflag.anonymous; import com.eastflag.listener.ClickListener; public class q2 { public static void main(String[] args) { /* * Anonymous class 문법으로 인스턴스 생성후 * onClick()을 호출하여 "clicked"를 출력되게 하라. */ } }
3. Anonymous class 문법으로 생성후 아래 함수 호출
package com.eastflag.anonymous; import com.eastflag.listener.ClickListener; public class q3 { public static void main(String[] args) { /* * 익명클래스로 인스턴스 생성후 * 아래 setClickListener 함수를 호출하여 "clicked" 출력하도록 하시오. */ } public static void setClickListener(ClickListener l) { l.onClick(); } }
4. 람다식으로 인스턴스를 생성
package com.eastflag.anonymous; import com.eastflag.listener.ClickListener; public class q4 { public static void main(String[] args) { /* * 람다식으로 인스턴스 생성후 * onClick() 호출하여 "clicked" 출력 */ } }
